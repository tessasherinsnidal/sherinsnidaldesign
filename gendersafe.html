Help Phone

The Help Line is a function that allows the user to talk to a representative anonymously and discuss their situation and feelings, get information and figure out their options.


Daily Log

This is a function that allows the user to keep track of their daily situation and can provide a record to look back on and learn from and is a record if the situation requires the police.


Group Conversation

The Group Conversation is a function that allows the user to talk to a group of women who are dealing with a similar situation.


Friends

This is a function that allows the user to build up a group of women that they have found helpful and they want to keep in contact with.
